# ClimaView – Web Analytics Project

## 📌 Project Title
Analysis of User Behavior and Traffic for ClimaView Website

## 🔗 Live Website
https://clima-view-sigma.vercel.app

## 🚀 Project Overview
This project focuses on applying web analytics and digital marketing techniques to analyze user behavior and improve website performance.

## 🛠️ Tools Used
- Google Analytics (GA4)
- Microsoft Clarity
- Google Search Console
- SEO (On-page & Off-page)
- Google Ads
- SendPulse (Email Marketing)

## 📊 Key Features
- User behavior tracking (heatmaps, recordings)
- Traffic analysis
- SEO optimization
- Campaign-based traffic generation
